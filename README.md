# css-comb package

CSS Comb for Atom.

[CSSComb](https://github.com/csscomb/csscomb.js)

## Usage

Press `ctrl+alt+c` to format you styles.

Plugin searches for `.csscomb.json` until it has been found and if it hasn't plugin takes default css-comb config.
